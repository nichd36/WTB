# Where's the Bus? (WTB)

A project which aimed to provide commuter with a better commuting experience.

# Background

As someone without a car living in Klang Valley, commuting by MRT/LRT/Monorail is convenient, however, things may get tricky when the destination that I'd like to visit to is not walkable from any station and only has bus connection.

PULSE app by Prasarana and Bus Kiosk works, but the user experience tends to be not satisfactory. WTB aims to solve this issue.
